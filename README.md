# CatFace

## 说明

🐱 喵脸识别

🐾 你发现一只猫，扫一扫~

🍾 恭喜你集齐一个猫图鉴！

👑 第一个发现新猫的人拥有给它命名的权利。

❤ **猫永远是对的！** 

![Cat's Never Wrong!](https://github.com/El-Chiang/CatFace/blob/master/images/CatIsNeverWrong.jpg?raw=true)



## 日志

- 2018-05-04
  - 收集猫片。

- 2018-05-05
  - 利用openCV识别出猫脸。
  - 裁剪猫脸图片，建立训练数据集。

- 2018-05-10

  - 数据预处理

- 2018-05-11

  - 使用`Keras`的CNN做猫脸图片分类，精度达到$83.3\%$以上。
